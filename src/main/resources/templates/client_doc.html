Computer alchemist v 1.0 API Documentation

HTTP Method URI Address Parameters Description
GET /components/{component}/{id}
  • @param `component` define type of Computer Component.
  • @param `id` define ID number of Component in database.
    		•

    You can fetch the JSON data of the component you are interested in.
    Obtained JSON contains HAL links to other available actions associated with component.


    Example: /components/motherboard/4225

    GET /components/{component}
  • @param `component` define type of Computer Component.
    		•

    You can fetch list of Components.
    Obtained JSON contains HAL links to other available actions associated with component.


    Example: /components/motherboard

    DELETE /components/{component}/{id}
  • @param `component` define type of Computer Component.
  • @param `id` define ID number of Component in database.
    		•

    You can remove Component from database if you have relevant credential (ADMIN).
    Obtained HttpStatus inform you what action has done.


    Example: /components/motherboard/4225

    POST /components/{component}/{id}/opinions
  • @param `component` define type of Computer Component.
  • @param `id` define ID number of Component in database.
    		•

    Each user can add opinion to product.
    Obtained JSON contains HAL links to other available actions associated with component.


    Example: /components/motherboard/4225/opinions

    GET /components/{component}/{id}/opinions
  • @param `component` define type of Computer Component.
  • @param `id` define ID number of Component in database.
    		•

    You can get opinions about the product.
    Obtained JSON contains HAL links to other available actions associated with component.


    Example: /components/motherboard/4225/opinions

    POST /components

    At this address you can post Component in correctly JSON format. Notice that @param `componentType`
    is most important at that because based on it Alchemist decides where to persist document.


    Example: /components
    Basic JSON Component example:

    {"producent":"AMD",
    "model":"Ryzen 5 1600",
    "description":"Description text...",
    "compatibleIndex":7.0,
    "componentType":"cpu",
    "cpuParameters":
    {"cores":6,
    "series":"Ryzen 5",
    "socket":"AM4",
    "threads":12,
    "bitArchitecture":"64bit",
    "cache":16,
    "frequency":3.2}}
